## Rules applied

- No independent “Premium Mosaike” category.
- Mosaics appear only as coordinated complements to the parent ceramic collection.
- A reference is not invented when it varies by format or finish; the public copy uses wording such as `Referenz je Format`.
- Products without an exact room/product image pair use a clearly product-led gallery instead of an unrelated room scene.
- AI-created scenes are only allowed inside Raumkonzepte and must be labelled `Raumkonzept – Visualisierung`.
